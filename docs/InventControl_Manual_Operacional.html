Logotipo (imagem) Arquivo PNG, JPG, WebP ou SVG com até 2 MB. Quando carregado, substitui o ícone Bootstrap. Recomendado: fundo transparente, altura ≥ 48 px.

Como trocar o nome e o ícone

  1. Acesse Admin → Tema no menu lateral
  2. Edite o campo Nome da empresa
  3. Opcionalmente, altere o campo Ícone Bootstrap — o preview ao lado atualiza em tempo real
  4. Clique em Salvar nome e ícone → a página recarrega com o novo nome

Como carregar um logotipo personalizado

  1. No card Logotipo (imagem), clique em Enviar novo logotipo e selecione o arquivo
  2. Um preview do arquivo aparece imediatamente na página
  3. Clique em Enviar logotipo para confirmar o upload
  4. A página recarrega e o logotipo passa a aparecer na barra lateral de todas as telas
  5. Para remover o logotipo e voltar ao ícone, clique em Remover logotipo
✅ A alteração é imediata para todos os usuários conectados — não é necessário reiniciar o servidor.
ℹ️ O logotipo fica armazenado em app/static/logos/logo.<ext> no servidor. Em instalações Docker, adicione um volume para persistir o arquivo entre reinicializações.

10. Alterar Senha

Qualquer usuário pode alterar sua própria senha clicando no ícone de chave 🔑 ao lado do nome de usuário no rodapé esquerdo do menu lateral.

http://192.168.1.100:8080/dashboard
🔑 Alterar Senha
✕
Cancelar Salvar

Regras

  • A senha atual deve estar correta
  • A nova senha deve ter pelo menos 6 caracteres
  • Nova senha e confirmação devem ser iguais
  • Após salvar, o modal fecha automaticamente em 1,5 s
  • Administradores podem redefinir a senha de outros usuários em Admin › Usuários Web › 🔑

Ao ligar, o terminal:

  1. Conecta ao WiFi configurado
  2. Busca a lista de operadores e turnos na API
  3. Verifica atualizações OTA
  4. Exibe a tela ociosa
⚠️ Se o WiFi falhar, o display exibe o erro e o terminal fica aguardando. Verifique a rede e reinicie.
